# Testing Policy

This root file is the quick entrypoint for running tests.

Detailed test architecture, suite breakdown, and coverage guidance now lives in:

- [docs/testing.md](./docs/testing.md)

## Quick Commands

```bash
npm run lint
npm run build
npm test -- --runInBand
```

Focused suites:

```bash
npm run test:unit
npm run test:integration
```
